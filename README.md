## 📱 Overview

InSec is a powerful personal safety application designed to provide quick access to help in critical situations while maintaining minimal battery usage.

Whether you need urgent help, companionship on your way home at night, access to defibrillators or first aid kits, or simply peace of mind — InSec connects you to your trusted network instantly.

Each user is assigned a unique code available in their profile. By exchanging codes, users can build a private network of trusted contacts (family, friends, caregivers, etc.).

When an alarm is triggered — via button press, volume buttons, device shake, or voice command — an alert is instantly sent to all members of the user’s safety network.

This project was fully designed and developed by me as a complete mobile solution.

## 🚀 Features

* 🔐 Secure communication & encrypted data
* 🚨 Emergency alert system (multi-trigger activation)
* 👥 Private safety network (invite-based system)
* 📍 Real-time location sharing
* 💬 Chat functionality
* 🎤 Voice commands & text-to-speech
* 📊 Data visualization
* 🔄 OTA updates (CodePush)
* 📱 Cross-platform (Android / iOS)
